# Selenium Cucumber Test Automation Framework

This project demonstrates an automated UI testing framework using **Selenium WebDriver** with **Cucumber** for Behavior-Driven Development (BDD). It supports clear Gherkin-based test cases and clean Page Object Model (POM) structure for maintainable tests.
